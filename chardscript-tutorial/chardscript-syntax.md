# ChardScript Syntax

## Quotes

In ChardScript, this programming language only accepts strings with a double quote, not a single quote.

#### Example

{% code lineNumbers="true" %}
```lua
echo("Hello World!")
```
{% endcode %}

ChardScript will give an error if it's like this.

#### Example

Syntax Error:

{% code lineNumbers="true" %}
```actionscript
echo('Hello World!')
```
{% endcode %}
